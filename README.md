# CheatzCSE
CheatzCSE – A Centralized Academic Hub for BRACU CSE Students
CheatzCSE is a centralized academic platform designed to support students of the Computer Science and Engineering (CSE) department at BRAC University. The goal of the project is to create a single platform where students can easily access course materials, lecture videos, and collaborative learning tools. Many students often struggle to find organized academic resources, especially when materials are scattered across different platforms. CheatzCSE solves this problem by providing a structured and user-friendly environment where all learning resources are organized by semester, course, week, and topic.
One of the core features of CheatzCSE is the Video Library, which provides an organized archive of YouTube lectures. Videos are categorized according to semester, course, and topic so that students can easily navigate and watch lectures directly within the platform. The system embeds YouTube videos for smooth playback and convenient access.
Another important feature is the Resource Hub, which acts as a centralized storage space for academic materials such as lecture slides, PDF notes, assignments, and other study resources. These materials are organized per course so that students can quickly download or view the files they need for their studies.
The platform also includes a powerful Search System that allows students to search across all resources, lectures, and topics. By entering keywords, users can quickly find relevant materials without manually browsing through different courses.
CheatzCSE provides a Course Catalog where students can explore all available CSE courses along with descriptions and semester information. A Semester Filter allows users to switch between semesters and view only the relevant courses and materials. Additionally, Topic Tags help students find resources based on specific concepts such as recursion, SQL, or data structures.
To improve the learning experience, CheatzCSE includes several personal study tools. Students can bookmark important lectures or resources, track their learning progress, and write personal notes attached to lectures. The platform also offers a Watch Later Queue where users can save videos to view at a later time.
The system encourages collaboration through community features such as lecture comments, resource ratings, and peer summaries. Students can discuss lecture topics, ask questions, and contribute summaries to help others understand complex concepts.
CheatzCSE also provides content discovery features such as Related Resources, Most Popular This Week, and Recently Added sections. These features help students discover useful materials quickly, especially during exam preparation periods.
Overall, CheatzCSE aims to simplify academic resource management and create a collaborative learning environment for BRAC University CSE students.
